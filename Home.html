Card Image Recognition

In order to recognize the cards, a model was trained on a sample of over 800 playing cards which were manually labelled. Thanks to Teachable Machine makes this process is easy, intuitive and best of all, free!

The sample of cards was created from a combination of a Github repository (16 photos of each rank in identical backgrounds), and user webcam photos (50+ of each rank taken in various lighting). The sample does not provide a 100% confidence each time.

Card Counting


The card counting capability is carried out through a python function. The function takes an input of card values and adds them to the tally. The python function should next be integrated with the js file to pass values which are over 95% confident (over 1 second intervals)
